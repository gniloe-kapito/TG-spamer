# 📤 Telegram Spammer v1.1

Простая программа на Python + Tkinter для отправки сообщений в Telegram. Без лишней сложности — запустил и работает.

---

## ✨ Что умеет

| Функция | Описание |
|---------|----------|
| 🚀 **Массовая отправка** | Отправь хоть 1, хоть 100 сообщений одной кнопкой |
| 📝 **Текст и медиа** | Поддержка текста, фото, видео и файлов |
| 💬 **Встроенный чат** | Смотри ответы от бота в реальном времени |
| 🎨 ** интерфейс** | Тёмная тема, аккуратный дизайн, приятные цвета — всё на вайбе |
| ⏸️ **Контроль** | Пауза, остановка, прогресс-бар — всё под рукой |
| ⚠️ **Защита от бана** | Автоматическая пауза при лимитах Telegram |

---

```bash
# 1. Установи зависимости
```
